### Installtion 
1. Clone this repository:

   ```bash
   git clone https://github.com/MrSubha420/Full-STACK-RECIPE-APP.git
   cd RecipeAppBackend
   ```

2. Install dependencies
   ```bash
   npm install

   ```
3. Setup your config file. 
      - create a new file 📁 at config folder **config.env** 


4. Run the app
  ```bash
  npm run dev.
  ```
